Matrix Multiplication

$C = A \times B$

A has dimensions $L\times M$

B has dimensions $M \times N$

C has dimensions $L \times N$

Matrix multiplication: $C_{[i,k]} = \sum_{j}(A_{[i,j]}\times B_{[j,k]})$

  • Map Phase:
    • for each element (i,j) of A, emit $((i,k), i, j, A_{[i,j]})$ for $k \in [1, N]$
    • for each element (j,k) of B, emit $((i,k), j, k, B_{[j,k]})$ for $i \in [1,L]$
  • Reduce Phase:
    • key = (i,k)
    • value = $\sum_{j}(A_{[i,j]} \times B_{[j,k]})$

Two-phase Map/Reduce Matrix Multiply

  • 1st Map Function:
    • for each matrix element $A_{[i,j]}$, emit $((j), A, i, j, A_{[i,j]})$
    • for each matrix element $B_{[j,k]}$, emit $((j), B, j, k, B_{[j,k]})$
  • 1st Reduce Function:
    • for each value of $(i,k)$ which comes from $A$ and $B$, emit $((i,k), A_{i,j} \times B_{j,k})$
  • 2st Map Function:
    • let the pair $((i,k), A_{i,j} \times B_{j,k})$ pass through
  • 2st Reduce Function:
    • emit $((i,k), \sum values)$
